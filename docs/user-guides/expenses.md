# User Guide: Expenses

The **Expenses** tab allows you to track operating expenses that are not
directly tied to product costs (COGS). This is essential for calculating
**Net Profit** and overall business health.

## Accessing Expenses

Click **Expenses** in the left sidebar.

## Why Track Expenses?

Product cost (COGS) only tells part of the story. To understand true
profitability, you need to account for:

- **Advertising**: Amazon PPC, Meta Ads, Google Ads.
- **Software Subscriptions**: Fulcrum, accounting tools, etc.
- **Shipping & Logistics**: Non-item-specific shipping costs.
- **Operating Costs**: Rent, utilities, labor.

## Managing Expenses

### Adding an Expense

1. Click **+ Expense**.
2. Fill in:
   - **Description**: What the expense is for.
   - **Amount**: Total cost.
   - **Category**: Select from predefined categories (Advertising, Software,
     Shipping, Other) or create a new one.
   - **Date**: When the expense occurred.
   - **Recurring**: Mark if this is a recurring expense (monthly, annually).
3. Click **Save**.

### Viewing Expense Summary

The Expenses tab shows:

- **Total Expenses** for the selected period.
- **Breakdown by Category** (pie chart).
- **Monthly Trend** (line chart).

### Filtering Expenses

Use the date range picker and category filter to narrow down the view.

## Expense Categories

| Category     | Example                              |
| ------------ | ------------------------------------ |
| Advertising  | Amazon Sponsored Products campaign   |
| Software     | Monthly Fulcrum subscription         |
| Shipping     | Carrier pickup fees                  |
| Other        | Office supplies, miscellaneous       |

## Reports

Expenses are integrated into the **Dashboard** for overall P&L calculations:

```
Net Profit = Revenue - COGS - Marketplace Fees - Expenses
```
