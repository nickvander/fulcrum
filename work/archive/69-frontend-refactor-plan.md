# Frontend Refactor & Testing Plan

**Status**: In Progress

## Objectives
1.  **Separation of Concerns**: independent HTML and CSS files for all components.
2.  **Robust Testing**: Comprehensive unit tests using Vitest.

## Steps
1.  **Analysis**: Identify components with inline styles/templates.
2.  **Refactoring**: Move inline code to separate files.
3.  **Testing**: Write/Update `.spec.ts` files.
4.  **Verification**: Run all tests and verify UI.
